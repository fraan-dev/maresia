# MaresIA: Mapeamento Inteligente de Resíduos Costeiros 

O **MaresIA** é um projeto acadêmico desenvolvido por estudantes do IFRN Campus Canguaretama. Trata-se de uma solução tecnológica ágil baseada em ferramentas No-Code e Inteligência Artificial para combater o descarte inadequado de lixo nas praias do Rio Grande do Norte.

---

##  Problemática Ambiental
O acúmulo de resíduos sólidos (como plásticos e vidros) no litoral potiguar ameaça a biodiversidade marinha, gera contaminação por microplásticos e prejudica a economia do turismo local. O maior desafio enfrentado por órgãos públicos e mutirões de limpeza é a falta de dados centralizados e em tempo real para localizar os focos críticos de poluição.

##  A Solução
O sistema permite que cidadãos e turistas reportem focos de lixo de forma rápida através de um aplicativo web. O fluxo da solução funciona da seguinte forma:
1. **Captura:** O usuário tira uma foto do lixo e o aplicativo coleta as coordenadas geográficas (GPS) automaticamente.
2. **Classificação (IA):** A imagem é enviada para uma API de Inteligência Artificial, que analisa e identifica o tipo de resíduo (plástico, metal, vidro ou orgânico).
3. **Mapeamento:** Os dados alimentam uma planilha centralizada e geram um mapa interativo público que direciona as equipes de coleta e limpeza.

---

##  Tecnologias Utilizadas
Para garantir viabilidade e agilidade no desenvolvimento, o ecossistema do projeto foi construído utilizando:

* **Interface (Front-end):** Glide Apps / Bubble (Plataforma No-Code para mobile)
* **Banco de Dados:** Google Sheets
* **Automação de Fluxo:** Make.com
* **Inteligência Artificial:** API Google Gemini Vision / OpenAI (Visão Computacional)
* **Visualização de Dados (Dashboard):** Google Looker Studio
