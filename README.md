# MaresIA: Mapeamento Inteligente de Resíduos Costeiros 

O **MaresIA** é um projeto acadêmico desenvolvido por estudantes do IFRN Campus Canguaretama. 
Trata-se de uma solução tecnológica prática baseada em Inteligência Artificial para combater o descarte inadequado de lixo, focada especificamente nas praias de Baía Formosa e Barra de Cunhaú, no Rio Grande do Norte.

---

## Problemática Ambiental
O acúmulo de resíduos sólidos (como plásticos e vidros) no litoral de Baía Formosa e Barra de Cunhaú ameaça a biodiversidade marinha, gera contaminação por microplásticos e prejudica a economia do turismo local. O maior desafio enfrentado por órgãos públicos e mutirões de limpeza nessas regiões é a falta de dados centralizados e em tempo real para localizar os focos críticos de poluição.

## A Solução
O sistema permite que cidadãos e turistas reportem focos de lixo de forma rápida através de uma aplicação web direcionada a essas localidades. O fluxo da solução funciona da seguinte forma:
- **Captura:** O usuário tira uma foto do lixo e o sistema armazena a localização do registro.
- **Classificação (IA):** A imagem é processada por uma Inteligência Artificial para identificar o tipo de resíduo (plástico, metal, vidro ou orgânico).
- **Mapeamento:** Os dados geram um mapa que direciona as equipes de coleta e limpeza nas praias atendidas.
